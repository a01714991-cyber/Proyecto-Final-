Proyecto Final – Sistema de Inventario para Tienda Deportiva (FYNIX) 
Samuel Estrella Martínez

Problema a resolver:
Una tienda deportiva necesita llevar un control básico de sus productos. 
Cada tipo de producto (playeras, shorts, accesorios) tiene atributos distintos, 
por lo que es ideal usar Programación Orientada a Objetos para organizarlos de
nmanera correcta.

Este proyecto es la base del futuro sistema de inventario de la tienda.

*¿Qué hace el programa actualmente?
- Define 3 clases diferentes:
1. Playera
2. Shorts Deportivos
3. Accesorios 

*¿Cómo se usa?
El usuario simplemente ejecuta el programa.
El sistema crea los productos automáticamente y muestra sus características.

*¿Qué se agregará en siguientes avances?
- Menús para que el usuario pueda elegir acciones.
- Agregar más tipos de producto.
- Sistema de inventario (agregar, eliminar, buscar productos).
- Guardar productos en archivos.
- Cálculo de precios y descuentos.
- Listas o vectores para almacenar varios objetos.
- Clases padre e hijos en un sistema más avanzado.
