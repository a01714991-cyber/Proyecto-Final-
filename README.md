Este proyecto implementa un sistema básico de tienda en C++ utilizando programación orientada a objetos, donde se modelan productos como playeras, shorts deportivos y accesorios. El código emplea herencia a partir de una clase base abstracta, encapsulación mediante atributos privados y protegidos, y funciones virtuales para mostrar información específica de cada producto. Además, se integra el concepto de agregación mediante una clase Inventario, la cual administra un conjunto de objetos Producto almacenados en un vector y permite registrar y visualizar múltiples elementos creados por el usuario. Para compilar y ejecutar el programa se debe usar un compilador compatible con C++11 o superior; en macOS o Linux se utiliza el comando g++ tienda.cpp -o tienda y posteriormente ./tienda. El programa solicita datos al usuario para crear los tres tipos de productos, los agrega al inventario y finalmente muestra toda la información capturada. Este sistema es compatible con cualquier entorno que disponga de un compilador C++ moderno y no requiere librerías externas adicionales.
