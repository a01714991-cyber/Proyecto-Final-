Este proyecto implementa un sistema básico de tienda en C++ utilizando programación orientada a objetos, donde se modelan tres tipos de productos: playeras, 
shorts deportivos y accesorios. El código emplea herencia, encapsulación mediante atributos privados y protegidos, y el uso de métodos getters, setters 
y funciones virtuales para mostrar información específica de cada producto. Para compilar y ejecutar el programa se debe usar un compilador compatible con
C++11 o superior; en sistemas basados en Unix como macOS o Linux, la compilación se realiza con el comando g++ tienda.cpp -o tienda, y posteriormente se ejecuta
usando ./tienda. El algoritmo solicita datos al usuario para crear cada producto y luego muestra la información capturada. Este proyecto es compatible con 
cualquier sistema operativo que disponga de un compilador C++ moderno como GCC o Clang, y no requiere librerías externas adicionales.
